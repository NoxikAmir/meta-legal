About the Project

Facebook Video Scheduler is a lightweight automation tool developed by AmirNoxik / Noxik Media Inc to simplify publishing workflows. It automates downloading popular YouTube videos and schedules them for posting to Facebook Pages at predefined time slots (22:00, 01:00, 04:00 GMT+4).

This tool is intended for personal and professional publishing needs while staying compliant with Meta’s developer policies.

How It Works

The scheduler works in three main steps:

  • Extracts metadata and downloads the best-quality YouTube video + thumbnail.
  • Uploads the video to Facebook using the official graph-video.facebook.com API.
  • Schedules publishing at the chosen slot with the preferred thumbnail.

All access tokens are securely stored as environment variables. No personal data is collected beyond what’s strictly necessary for publishing.
